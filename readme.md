# SpaceDefender
SpaceDefender is a simple side-scrolling shooter developed in Java using Slick2D library.

## Features
- infinite number of enemies
- statictics stored in .txt file
- secret code giving extra 27 lives

## Controlls
- arrow keys - move
- spacebar - shoot
- escape - pause
- enter - select
